Java 基础笔记

Java 的程序结构

结构图


类存在于源文件里面
方法存在于类中
语句（statement）存在于方法中

基本概念

什么是源文件？
源文件（扩展名为.java）带有类的定义。类用来表示程序的一个组件，小程序或许只会有一个类。
什么是类？
类中带有一个或多个方法。方法必须在类中声明。
什么是方法
在方法的花括号中编写方法应该执行的命令。方法代码是由一组语句所构成，你可以把方法想象成一个函数或过程。

为什么所有的东西都包含在类中？
因为Java是面向对象的语言，Java中的绝大多数东西都是对象。当Java虚拟机启动执行时，它会寻找你在命令列所制定的类。然后它会锁定像下面这样一个特定的方法：

public static void main (String[] args){
//程序代码写在这里
}

接着Java虚拟机就会执行main方法在花括号间的函数所有的指令，每个Java程序最小都会有一个类以及一个main().每个应用程序只有一个mian()函数。

一简单的条件分支程序

class Test{
public static void main(String[] args){
  int x=3;
  if(x==3){
     System.out.printIn("x must be 3")
  }
  System.out.printIn("This runs no matter what");
 }
}
// System.out.printIn相较于System.out.print会在最后面插入换行，若你想要让后续的输出以新的一行开始，可以使用printIn。

输出：

x must be 3
This runs no matter what
