零基础运行别人写好的python 爬虫程序

By yao

发表于 2016-05-11

文章目录

使用的爬虫程序是桩白墨 PLOUGH的我的第一只爬虫：爬取豆瓣读书
1.先下载python
2.python目录添加到环境变量里
3.打开IDLE (Python GUI)可在开始里面查找
4.点击File->Open打开下载的book_list_spider.py文件，然后Run

报错：
1.ImportError:No module named request
解决方法是使用pip软件包管理工具进行安装和卸python包。pip的安装可以参照此链接windows下python包管理工具pip的安装与使用
pip的Scripts的路径添加到环境变量后，直接在window的命令窗口执行pip install request,下载完之后再次运行爬虫程序
2.ImportError: No module named ‘bs4’
在window的命令窗口执行pip install BeautifulSoup4
3.NameError: name ‘reload’ is not defined
在爬虫程序中添加from imp import reload，之后再运行会报ttributeError: ‘module’ object has no attribute ‘setdefaultencoding’的错误，在网上查找了解决方案，但是还是报这个错误，这句命令的作用是设置默认编码方式为utf-8,实在不知道为啥不起效果我就把它给注释了。
4.UnicodeEncodeError: ‘gbk’ codec can’t encode character ‘\u2022’ in position
是因为写入文件的编码方式为GBK,无法对汉字进行编码，所以报错，解决方法来至此链接http://blog.csdn.net/greatpresident/article/details/8209712
直接改变目标文件的编码f = open(file_name, ‘w’,encoding=’utf-8’)，之后再次运行程序就成功了，之后可在python安装目录里查看book-list.txt
